# Ubuntu Server VM Lab

## Goal

Set up a local Ubuntu Server virtual machine and apply basic security best practices.

## What I learn/able to do

- Installing and configuring a Linux server
- Understand user permissions and sudo
- The why and avoid using root directly
- Enable and verify SSH access
- Enable firewall on boot

## Environment

- Host OS: Windows 11
- Virtualization(hypervisor): Oracle VirtualBox
- Guest OS: Ubuntu Server (LTS) //not desktop\*

## Lab Overview

The Ubuntu Server runs inside VirtualBox.  
Administration is done using a non-root admin user with controlled sudo access.
Soon start running, managing and securing my first services on my server
